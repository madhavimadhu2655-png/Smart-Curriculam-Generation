<product_requirements>
The project aims to develop EduTrack, a comprehensive educational platform that integrates AI-powered learning path generation with a robust attendance management system. The initial phase involved merging a Streamlit-based Learning Path Generator (using YouTube, Notion/Drive via Composio) and a Flask-based Attendance Calculator into a FastAPI/React/MongoDB stack.

**Key features include:**
*   **Learning Path Generation:** AI-driven, day-wise learning paths with YouTube video links, topic names, and daily study times. Paths support both teachers assigning to students and individual self-learning.
*   **Progress Tracking:** Attendance-style monitoring of learning progress, tracking video completion and study time.
*   **User Roles:** Teacher (creates/assigns paths, monitors students) and Student (accesses paths, tracks progress).
*   **UI Redesign (EduTrack Pro/EduTrack):** A complete visual overhaul to match a specified demo website, featuring a modern, responsive design with specific components.
    *   **Dashboard:** Displays student/subject/attendance/path counts, weekly attendance charts, and recent activity.
    *   **Student Management:** CRUD for students, QR code generation/download, search/filter.
    *   **QR Scanner:** Real-time attendance marking via camera-based QR scanning, with instant updates.
    *   **Learning Paths:** Custom creation, progress tracking, Watch buttons for YouTube videos, and detailed schedule display.
    *   **Report Generation:** Monthly/yearly attendance reports, exportable as PDF/Excel.
    *   **Branding:** Renamed from EduTrack Pro to EduTrack.
</product_requirements>

<key_technical_concepts>
- **Full-stack Development:** React for the frontend, FastAPI for the backend, and MongoDB as the database.
- **AI/LLM Integration:** Langchain, Langgraph, ChatGoogleGenerativeAI (Gemini-2.5-flash), utilizing emergent integrations and an emergent LLM key.
- **External APIs:** YouTube Data API v3 for video search and content curation.
- **Authentication:** JWT-based system with role-based access control (Teacher/Student).
- **UI/UX:** Modern React patterns, Tailwind CSS for styling, responsive design, and glass morphism effects.
- **Deployment:** Containerized application running in a Kubernetes environment with Supervisor for process management.
</key_technical_concepts>
